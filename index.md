## Главная страница (точка входа на сайт)

**Метод:** GET  
**URL:** /

**Пример запроса на сервер:**  
https://educube.ru/

**Назначение:**  
- точка входа на сайт;

**Поведение:**  
- отображает главную страницу;
- отображает разделы каталога;
- отображает рекомендуемые товары;
- принимает заявки пользователей на обратную связь;

**Response:**  
200 OK

**Errors:**  
404 - not found;   
403 -  forbidden;   
500 -  server error;
